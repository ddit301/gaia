### Gaia is how we develop together.

기존의 Project Management System들의 어려운 사용방법과 높은 진입장벽을 해결하기 위해 기획했습니다.<br/>
프로젝트 관리와 개발자간의 협업을 돕습니다.<br/>
> 아래의 기능들을 제공합니다

- 코드 저장소
- 이슈 트래킹
- 프로젝트 일정 관리
- 인스턴트 메신저
- 클라우드 스토리지
- Web IDE
